# Story 1.8: Sinais Vitais com Timeline

**Status:** Done
**Sprint:** 1
**Assignee:** @dev
**Points:** 5

## Descricao
Como profissional de saude, quero registrar e visualizar sinais vitais em uma timeline agrupada por data, para que eu possa acompanhar a evolucao clinica do paciente ao longo do tempo.

## Criterios de Aceitacao
- [x] Timeline agrupada por data com linha vertical conectando entradas
- [x] Formulario para novo registro: PA, FC, Temperatura, SatO2, autor, data/hora
- [x] Codificacao por cor: FC >120 ou <50 em vermelho, Temp >38 em vermelho, SatO2 <94 em vermelho
- [x] Exclusao de registro com confirmacao
- [x] Dados persistidos e ordenados cronologicamente

## Arquivos Modificados
- `src/lib/supabase/vitals.ts`
- `src/app/(dashboard)/pacientes/[id]/vitais/page.tsx`

## Notas Tecnicas
Timeline implementada com agrupamento por data usando reduce. Linha vertical via CSS (border-left). Thresholds de alerta configurados como constantes para facil ajuste. Formulario usa valores padrao de data/hora atual.
